# 易题通 MVP 功能总览文档

## 1. 文档信息
- 产品：易题通（MVP）
- 技术栈：Spring Boot + MyBatis + MySQL + Vue3 + TypeScript + Element Plus
- 文档目的：说明当前版本已实现功能、接口能力、数据流与边界，便于演示、验收与后续迭代。

## 2. 系统整体说明
当前版本已完成核心闭环：
- 题目浏览
- 在线编程
- 运行测试
- 正式提交
- 判题回写
- 错题归档
- AI解释/建议
- 推荐与推荐记录

架构上分为：
- 前端用户端
- 前端管理端
- 后端业务服务
- 判题执行模块（当前以 Java 主链路为主）

## 5. 判题流程（当前实现）

### 5.1 提交流程
1. 创建提交记录，状态 PENDING
2. 进入判题状态 JUDGING
3. 编译代码
4. 编译通过后逐测试点执行
5. 写入测试点结果明细
6. 聚合最终状态并回写 submission

### 5.2 最终状态判定
- 编译失败 -> COMPILE_ERROR
- 运行超时 -> TIME_LIMIT_EXCEEDED
- 运行异常 -> RUNTIME_ERROR
- 输出不一致 -> WRONG_ANSWER
- 全部通过 -> ACCEPTED

### 5.3 后处理
- 非 ACCEPTED 提交进入 wrong_questions
- 可触发 AI 解释
- 推荐支持手动生成

## 9. 当前版本边界与约束

### 9.2 资源联动策略
- 当前版本已暂停“资源联动展示”方向

### 9.3 安全与工程

- JWT 鉴权

## 10. 演示建议流程（5-8分钟）
1. 注册/登录普通用户
2. 题库筛选并进入题目
3. 运行测试与正式提交
4. 提交记录查看状态流转与明细
5. 失败后查看错题本
6. 手动生成推荐并查看推荐记录
7. 进入学习数据页，手动生成学习建议
8. 切管理员账号演示管理端（用户/题目/标签/监控）
